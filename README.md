# Heart Rate Monitor Simulation

This project simulates heart rate measurement using a synthetic ECG-like signal.
It generates a waveform, adds noise, filters the signal, and calculates heart rate
in beats per minute (BPM).

## Features
- Simulated ECG waveform
- Noise addition and filtering
- Peak detection to calculate BPM
- Matplotlib visualization
- Exported CSV data

## Requirements
Python 3.x  
Install dependencies using:
pip install -r requirements.txt

## How to Run
1. Clone the repository:
git clone https://github.com/YOURUSERNAME/heart-rate-monitor-simulation

2. Install dependencies:
pip install -r requirements.txt

3. Run the script:
python src/heart_rate.py
